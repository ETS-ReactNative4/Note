# random.choice\(\)

import random

random.choice\(seq\)

choice\(\) 方法返回一个列表，元组或字符串的随机项。

seq -- 可以是一个列表，元组或字符串。

```text
print "choice([1, 2, 3, 5, 9]) : ", random.choice([1, 2, 3, 5, 9])
print "choice('A String') : ", random.choice('A String')

choice([1, 2, 3, 5, 9]) :  2
choice('A String') :  n
```
