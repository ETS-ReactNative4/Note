## Redis Select
redis默认有db0~db15之多。
Redis Select 命令用于切换到指定的数据库，数据库索引号 index 用数字值指定，以 0 作为起始索引值。默认库为0
```
127.0.0.1:6379> select 5
OK
127.0.0.1:6379[5]> keys *
1) "queue:job:1"
2) "queue:jobs:inactive"
3) "queue:job:types"
4) "queue:job:2"
5) "queue:ids"
6) "queue:jobs"
7) "queue:order/line-0:jobs"
8) "queue:jobs:order/line-0:inactive"
```
