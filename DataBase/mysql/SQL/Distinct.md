## 语法
SELECT DISTINCT 列名称 FROM 表名称

有mark表，如下
```
mysql> select * from mark;
+----+--------+--------+-------+
| id | name   | lesson | score |
+----+--------+--------+-------+
|  1 | 张三   | 语文   |    81 |
|  2 | 张三   | 数学   |    65 |
|  3 | 李四   | 语文   |    23 |
|  4 | 李四   | 数学   |    90 |
|  5 | 王五   | 语文   |    94 |
|  6 | 王五   | 数学   |    81 |
|  7 | 赵六   | 数学   |    80 |
|  8 | 赵六   | 英语   |    86 |
+----+--------+--------+-------+
8 rows in set (0.01 sec)
```

选出name 列
```
mysql> select name from mark;
+--------+
| name   |
+--------+
| 张三   |
| 张三   |
| 李四   |
| 李四   |
| 王五   |
| 王五   |
| 赵六   |
| 赵六   |
+--------+
8 rows in set (0.00 sec)
```

重复的名字支取一次
```
mysql> select distinct name from mark;
+--------+
| name   |
+--------+
| 张三   |
| 李四   |
| 王五   |
| 赵六   |
+--------+
4 rows in set (0.00 sec)
```
