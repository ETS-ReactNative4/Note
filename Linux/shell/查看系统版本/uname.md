Shell中uname命令查看系统内核、版本

## 用于打印内核名称和版本、主机名等系统信息
```
[root@test ~]# uname
Linux

[root@test ~]# uname -a
Linux test.xuptip.com 3.10.0-123.el7.x86_64 #1 SMP Mon May 5 11:16:57 EDT 2014 x86_64 x86_64 x86_64 GNU/Linux

在window git bash下：
Admin@MXJ-PC MINGW64 /e/Joy/Note/Linux/shell (master)
$ uname
MINGW64_NT-10.0

Admin@MXJ-PC MINGW64 /e/Joy/Note/Linux/shell (master)
$ uname -a
MINGW64_NT-10.0 MXJ-PC 2.10.0(0.325/5/3) 2018-07-25 13:06 x86_64 Msys
```
