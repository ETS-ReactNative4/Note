The cursor CSS property sets mouse cursor to display when the mouse pointer is over an element.
https://developer.mozilla.org/en-US/docs/Web/CSS/cursor
cursor: help;
cursor: wait;
cursor: crosshair;
cursor: not-allowed;
cursor: zoom-in;
cursor: grab;
custom
/* URL, with a keyword fallback */ cursor: url(hand.cur), pointer; /* URL and coordinates, with a keyword fallback */ cursor: url(cursor1.png) 4 12, auto; cursor: url(cursor2.png) 2 2, pointer;
